# The Six Laws

In June 2026 I recorded a conversation with Alex Pedori, an AI engineer, about the agent system I built to run my own business. I am not an engineer. I am a coach and a public speaker, and I built thirteen AI agents to find and follow up on paid speaking gigs.

On the recording I promised to share the documents we talked about, so anyone can hand them to their own AI agents. Here they are.

## What's inside
- **SIX-LAWS-v1.0.md** — the six rules I gave my agents. Don't lie to me. Don't waste my time. Write it down. Improve. Make money. The constitution every agent loads first.
- **SYSTEM-COMMENTARY.md** — how the system actually works, and the hard lesson behind each law.
- **SHOW-NOTES.md** — the episode notes: the tools we named, a short glossary, and the running jokes.
- **TRANSCRIPT.md** — the full conversation, cleaned and labeled by speaker.

## How to use it
Drop these files into your own AI agent and tell it to read them. The Six Laws are written to be loaded by an agent at the start of every session.
