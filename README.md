# The Six Laws

In June 2026, Matteo Cassese, a keynote speaker and coach, sat down with Alex Pedori, a senior AI engineer, to talk through the agent system Matteo built to run his own business. Matteo is not an engineer. He is a coach and a public speaker, and he built thirteen AI agents to find and follow up on paid speaking gigs.

On the recording, Matteo promised to share the documents the two of them discussed, so anyone can hand them to their own AI agents. They are collected here.

## What's inside
- **SIX-LAWS-v1.0.md**: the six rules Matteo gave his agents. Don't lie to me. Don't waste my time. Write it down. Improve. Make money. The constitution every agent loads first.
- **SYSTEM-COMMENTARY.md**: how the system actually works, and the hard lesson behind each law.
- **SHOW-NOTES.md**: the episode notes, the tools named in the conversation, a short glossary, and the running jokes.
- **TOOLS-AND-STACK.md**: the full stack of tools the system runs on.
- **TRANSCRIPT.md**: the full conversation, cleaned and labeled by speaker.

## How to use it
Drop these files into an AI agent and tell it to read them. The Six Laws are written to be loaded by an agent at the start of every session.
